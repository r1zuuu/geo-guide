# Geo Guide

## Overview

Geo Guide is a web application built with Next.js and TypeScript that supports geography-based learning through interactive map content.

The project combines educational functionality with geospatial tools, offering an engaging way to explore maps and location-based information.

## Features

- interactive map-based interface
- educational geography content
- multiple difficulty levels
- support for geospatial interaction
- structured frontend architecture
- backend integration with Supabase

## Tech Stack

- Next.js
- React
- TypeScript
- Leaflet
- React Leaflet
- Supabase
- Tailwind CSS

## Getting Started

To run the project locally:

```bash
npm install
npm run dev
